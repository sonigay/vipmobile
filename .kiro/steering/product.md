# Product Overview

## (주)브이아이피플러스 - VIP Map Application

A comprehensive web application for managing mobile phone inventory, sales, and store operations across multiple retail locations.

## Core Purpose

The system provides real-time inventory tracking, sales management, and operational tools for mobile phone retail stores and their management hierarchy (agents, direct stores, administrators).

## Key Features

- **Map-based inventory visualization** using OpenStreetMap/Leaflet
- **Multi-mode operation** with role-based access control (20+ operational modes)
- **Real-time inventory management** with assignment and recovery workflows
- **Sales tracking and activation data** with monthly comparisons
- **Settlement and inspection workflows** for quality control
- **Policy management and document distribution**
- **Meeting coordination and reservation systems**
- **SMS management and automated responses**
- **Direct store operations** with customer queue management
- **Budget tracking and meal allowance management**

## User Roles

- **Agents (대리점)**: Manage multiple stores, view aggregated data
- **Store Users**: View and request inventory for their location
- **Inventory Managers**: Oversee stock allocation and audits
- **Settlement Managers**: Handle financial reconciliation
- **Direct Store Staff**: Customer-facing operations and queue management
- **Administrators**: System-wide configuration and oversight

## Data Sources

- Google Sheets API as primary data store
- Discord integration for logging and notifications
- Kakao Maps API for geocoding
- Web push notifications for real-time alerts
