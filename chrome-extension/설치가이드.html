📥 2단계: 확장 프로그램 다운로드

⭐ 방법 1: VIP 앱에서 다운로드 (가장 쉬움!)

VIP 앱 접속
https://vipmobile.netlify.app 에 접속하세요
다운로드 버튼 클릭
로그인 화면의 주황색 박스에서
"📥 2단계: VIP 확장 프로그램 다운로드" 버튼 클릭
ZIP 파일 다운로드
vip-chrome-extension.zip 파일이 자동으로 다운로드됩니다
압축 해제
다운로드 폴더에서 ZIP 파일 찾기
• Windows: 우클릭 → "압축 풀기"
• Mac: 더블 클릭
폴더 확인
압축 해제된 폴더를 바탕화면이나 쉽게 찾을 수 있는 곳에 이동

🔧 3단계: Chrome 확장 프로그램 설치

확장 프로그램 관리 페이지 열기
Chrome 주소창에 입력:
chrome://extensions/
또는 Chrome 우측 상단 ⋮ → 도구 더보기 → 확장 프로그램
개발자 모드 활성화
우측 상단의 "개발자 모드" 토글을 켜기(ON)로 변경
확장 프로그램 로드
좌측 상단 "압축해제된 확장 프로그램을 로드합니다" 클릭
→ 압축 해제한 폴더 선택 → "폴더 선택"
설치 완료 확인
확장 프로그램 목록에 "VIP 대리점 정보 보호" 표시
상태가 "켜짐"인지 확인

✅ 4단계: 작동 확인

VIP 앱 새로고침
VIP 앱 페이지에서 F5 키를 눌러 새로고침
로그인 시도
P코드를 입력하고 로그인 버튼 클릭
성공!
로그인이 성공하면 설치 완료입니다! 🎉
💡 확장 프로그램 작동 확인:
U+ 가입 페이지 접속 시 우측 상단에 "🔒 정보 보호 활성화" 메시지가 3초간 표시됩니다

🔧 문제 해결

❌ "확장 프로그램이 필요합니다" 에러

원인: 확장이 설치되지 않았거나 비활성화됨

해결:
1. chrome://extensions/ 접속
2. "VIP 대리점 정보 보호" 찾기
3. 토글이 파란색(켜짐)인지 확인
4. 꺼져 있으면 켜기
5. VIP 앱 페이지 새로고침(F5)
6. 다시 로그인 시도
❌ "Chrome 브라우저만 사용 가능" 에러

원인: Chrome이 아닌 다른 브라우저 사용 중

해결:
1. Chrome 브라우저 다운로드: https://www.google.com/chrome/
2. 설치 후 Chrome에서 VIP 앱 접속
❌ 확장 프로그램이 목록에 없음

해결:
1. 설치 과정을 처음부터 다시 시도
2. 올바른 폴더를 선택했는지 확인
3. manifest.json 파일이 폴더 안에 있는지 확인

💡 참고사항

빠른 접근을 위한 확장 고정

  1. Chrome 우측 상단 확장 아이콘(퍼즐 모양) 클릭
  2. "VIP 대리점 정보 보호" 옆의 핀 아이콘 클릭
  3. 주소창 옆에 항상 표시됨

확장 프로그램 업데이트 방법

  1. 기존 확장 제거 (chrome://extensions/에서 "제거")
  2. 새 버전 다운로드 (VIP 앱에서)
  3. 위의 설치 방법과 동일하게 진행
