🔧 VIP 필수 확장프로그램

5분 안에 설치 완료!

⚠️ 필수 설치
VIP 앱의 매장 사용자는 반드시 이 확장 프로그램을 설치해야 합니다.

📥 1단계: 확장 프로그램 다운로드

페이지 새로고침
VIP 앱 페이지에서 Ctrl+F5 키로 새로고침
다운로드 버튼 클릭
로그인 화면에서 "📥 VIP 확장 프로그램 다운로드" 버튼 클릭
압축 해제
다운로드된 ZIP 파일을 압축 해제하세요
• Windows: 우클릭 → "압축 풀기"
• Mac: 더블 클릭

🔧 2단계: Chrome/Edge에 설치

Chrome 사용 시:

확장 프로그램 페이지 열기
Chrome 주소창에 입력:
chrome://extensions/
개발자 모드 켜기
우측 상단의 "개발자 모드" 토글을 ON으로 변경
확장 로드
좌측 상단 "압축해제된 확장 프로그램을 로드합니다" 클릭
→ 압축 해제한 폴더 선택

Edge 사용 시:

확장 프로그램 페이지 열기
Edge 주소창에 입력:
edge://extensions/
개발자 모드 켜기
좌측 하단의 "개발자 모드" 토글을 ON으로 변경
확장 로드
"압축을 푼 확장 로드" 클릭
→ 압축 해제한 폴더 선택
✅ 설치 완료:
목록에 "VIP 필수 확장프로그램" 표시 확인

✅ 3단계: 로그인

페이지 새로고침
VIP 앱 페이지에서 Ctrl+F5 키로 새로고침
로그인
P코드를 입력하고 로그인
완료!
로그인 성공 시 설치 완료 🎉

🔧 문제 해결

❌ "확장 프로그램이 필요합니다" 에러

해결:
1. chrome://extensions/ 접속
2. 확장이 켜짐 상태인지 확인
3. 꺼져 있으면 켜기
4. 페이지 새로고침 후 다시 로그인
❌ "Chrome 또는 Edge 브라우저만 사용 가능" 에러

해결:
Chrome 또는 Edge 브라우저를 사용하세요
