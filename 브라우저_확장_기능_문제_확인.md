# 브라우저 확장 기능 문제 확인 및 해결 방법

## 문제 상황
Cursor의 브라우저 확장 기능 MCP 서버가 작동하지 않아 `browser_navigate`, `browser_snapshot` 등의 도구를 사용할 수 없습니다.

## 원인 분석

### 1. MCP 서버 미설정
- Cursor의 브라우저 확장 기능은 MCP (Model Context Protocol) 서버를 통해 작동합니다.
- `list_mcp_resources` 결과가 비어있어 MCP 서버가 설정되지 않았거나 연결되지 않은 것으로 보입니다.

### 2. 가능한 원인들
1. **Cursor 설정에서 MCP 서버 미활성화**
   - Cursor 설정 → Features → MCP Servers에서 브라우저 확장 기능이 활성화되지 않았을 수 있습니다.

2. **브라우저 확장 프로그램 미설치**
   - Cursor 브라우저 확장 프로그램이 Chrome/Edge에 설치되지 않았을 수 있습니다.

3. **MCP 서버 연결 실패**
   - 네트워크 문제나 권한 문제로 MCP 서버에 연결할 수 없을 수 있습니다.

## 해결 방법

### 방법 1: Cursor 설정 확인
1. Cursor 설정 열기 (Ctrl+, 또는 Cmd+,)
2. "MCP" 또는 "Model Context Protocol" 검색
3. 브라우저 확장 기능 MCP 서버가 활성화되어 있는지 확인
4. 활성화되어 있지 않다면 활성화

### 방법 2: Cursor 브라우저 확장 프로그램 설치
1. Chrome/Edge 브라우저 열기
2. Cursor 공식 문서에서 브라우저 확장 프로그램 다운로드 링크 확인
3. 확장 프로그램 설치
4. Cursor 재시작

### 방법 3: 수동으로 이미지 확인
브라우저 확장 기능이 작동하지 않는 경우, 사용자가 직접 Discord 이미지 URL을 브라우저에서 열어 확인할 수 있습니다.

## 대안: 이미지 URL 직접 확인
브라우저 확장 기능이 작동하지 않을 때는 다음 방법을 사용할 수 있습니다:

1. **Discord 이미지 URL 직접 열기**
   - 제공된 Discord 이미지 URL을 브라우저 주소창에 붙여넣기
   - 이미지가 로딩 화면인지 실제 데이터 화면인지 확인

2. **브라우저 개발자 도구 사용**
   - F12로 개발자 도구 열기
   - Network 탭에서 이미지 로드 확인
   - Console 탭에서 에러 확인

## 참고사항
- 브라우저 확장 기능은 Cursor의 선택적 기능입니다.
- 이미지 확인은 수동으로도 가능합니다.
- 코드 수정은 브라우저 확장 기능 없이도 가능합니다.
